## BPMN — the process lens

Design and edit **BPMN 2.0** process diagrams as a native custom editor — model how
work actually flows, right next to your code.

Files use the `.bpmn` extension (DMN decision tables use `.dmn`) and open in the
BPMN modeler.

**Open the BPMN tutorial** (the button on the left) for a guided walkthrough —
create, learn the basics, and deploy — from the modeler itself.
