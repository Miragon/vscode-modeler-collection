# Miragon Modeler Collection

**Model-based design in VS Code — BPMN, Team Topologies, Wardley Maps and Domain Storytelling in a single install.**

This is an [Extension Pack](https://code.visualstudio.com/api/references/extension-manifest#extension-packs):
one click installs the whole Miragon modeler family, so you can move between process, organization,
strategy and domain diagrams without leaving your editor.

## What's inside

- **[BPMN Modeler](https://marketplace.visualstudio.com/items?itemName=miragon-gmbh.vs-code-bpmn-modeler)** —
  design and edit BPMN 2.0 process diagrams as a native VS Code custom editor.
- **[Team Topologies Modeler](https://marketplace.visualstudio.com/items?itemName=miragon-gmbh.team-topologies-modeler)** —
  map team types and interaction modes to visualize and evolve your organization.
- **[Wardley Mapping Modeler](https://marketplace.visualstudio.com/items?itemName=miragon-gmbh.wardley-mapping-modeler)** —
  build Wardley Maps to reason about strategy along the value chain and evolution axis.
- **[Egon.io — Domain Story Modeler](https://marketplace.visualstudio.com/items?itemName=miragon-gmbh.egon-vscode-extension)** —
  capture domain stories to understand a business domain in the language of the people who live it.

## Why a collection?

Each modeler is a different lens on the same system, and they're strongest together:

- **Process** — how work flows (BPMN)
- **Organization** — who does the work and how teams interact (Team Topologies)
- **Strategy** — where to invest and how components evolve (Wardley Maps)
- **Domain** — what the business actually does, told as stories (Egon.io)

Installing them as a set means the whole toolkit is one command away when you switch perspectives.

## Standalone by design

Every modeler is also installable on its own and ships on its **own release cycle** — this pack
simply bundles the current family for convenience. Install only what you need, or take the set.
